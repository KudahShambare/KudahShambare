## ⚡ Engineering Philosophy & Current Explorations
My engineering focus is on architecting complete web solutions, from responsive, high-performance frontends to secure serverless backends. As an MTech candidate in Cloud Computing and a technical founder, my research and professional interests are deeply rooted in scalable **e-commerce systems, multi-sided marketplaces, and enterprise-grade cloud infrastructure**.

Beyond standard web development, I am heavily invested in solving high-friction connectivity challenges.
